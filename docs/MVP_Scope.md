# MVP Scope (Minimum Viable Product)

## What MUST Work (Version 1)

### For Students:
- Login with email/password
- See study materials (PDFs uploaded by teacher)
- Ask AI questions (get answers from PDFs)
- Take quizzes (auto-generated by AI)
- See their scores

### For Teachers:
- Login
- Upload PDF files
- Click "Generate Quiz" button
- See student quiz results

### For Principal:
- Login
- See dashboard with:
  - Total students
  - Average scores
  - Weakest subjects
  - AI insights

## What Will NOT Be in Version 1

- Mobile app (will be website only)
- Parent login
- Attendance system
- Live video classes
- Payment system
- Report card generation

## Timeline

- Week 1-2: Setup + Login system
- Week 3: PDF upload + AI reading
- Week 4: Quiz generation
- Week 5: Analytics dashboard
- Week 6: Testing + Launch
