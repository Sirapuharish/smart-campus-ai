# How Users Will Use SmartCampus

## Student Journey

1. Student opens website
2. Types email and password
3. Clicks "Login"
4. Sees Dashboard with 4 options:
   - Study Material
   - Ask AI
   - Take Quiz
   - My Performance

## Teacher Journey

1. Teacher logs in
2. Sees Dashboard with 3 options:
   - Upload PDF
   - Generate Quiz
   - View Results

## Principal Journey

1. Principal logs in
2. Sees Dashboard with charts and numbers
3. Reads AI insights like "Class 8 needs help in Algebra"
