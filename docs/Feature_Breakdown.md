# Feature Breakdown

## Student

### Login
Description:
Student logs into the system.

### Ask AI
Description:
Student asks questions from uploaded study materials.

### Quiz
Description:
Student takes AI-generated quizzes.

### Performance
Description:
Student views scores and progress.

--------------------------------

## Teacher

### Upload PDF
Description:
Teacher uploads study materials.

### Generate Quiz
Description:
AI generates quiz questions from uploaded content.

### View Results
Description:
Teacher views student performance.

--------------------------------

## Principal

### Dashboard
Description:
View institution statistics.

### Analytics
Description:
View performance trends and weak subjects.
